Installation
------------


Load google-visualization-php into your project by adding the following lines to your `composer.json`

``` json
{
    "require": {
        "vube/google-visualization-php": ">=0.1"
    }
}
```


Dependencies
------------

- PHP 5.3.2+
- Composer
